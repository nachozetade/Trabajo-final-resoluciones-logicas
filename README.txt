Veterinaria C.C.C — Sitio de E-commerce

Proyecto final desarrollado para la materia de Resoluciones Lógicas | Alicia Moreau De Justos | 4°4
El objetivo fue crear un sitio web funcional tipo e-commerce utilizando **HTML, CSS, JavaScript y Bootstrap**, aplicando los conceptos vistos durante el año.
Descripción del proyecto:
El sitio representa una veterinaria llamada "C.C.C Veterinaria", donde se pueden visualizar distintos productos para mascotas (camas, bowls, cuerdas, peines, etc.), agregar al carrito y simular una compra.

El diseño es funcional y coherente visualmente.  
Cada integrante colaboró en el desarrollo, maquetado, estilo y funcionalidad del sitio.
Tecnologías utilizadas:
- HTML5 → Estructura semántica del sitio  
- CSS3 → Estilos personalizados  
- Bootstrap 5 → Diseño adaptable a distintos dispositivos  
- JavaScript → Funcionalidades dinámicas del carrito y alertas  
 Estructura del proyecto:
/ecommerce
│
├── /assets → Imágenes e íconos utilizados
├── /css → Archivos de estilos (style.css)
├── /js → Archivos JavaScript (main.js)
├── /pages → Páginas internas (Nosotros, Productos, Info de productos, Carrito)
├── index.html → Página principal
└── README.md → Explicación del proyecto
